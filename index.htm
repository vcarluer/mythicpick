Mythic RPG random pick generator

This tool simulates the mythic chaos and question rolls done during a game.

It can be used too as an idea generator, it was initally built to do so: write adventure scenario for video games using mythic rules.

You can embed this generator in your web site: just add a reference to http://www.gamersassociate.com/mythicpick/mythicpick.js, http://www.gamersassociate.com/mythicpick/mythicpick.css and create a div with id="mythic" in your html.
